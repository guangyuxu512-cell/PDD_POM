2. 基础页增加随机延迟工具（pages/基础页.py）
新增一个通用方法，所有 Page 类继承后都能用：
import random
import asyncio

async def 随机延迟(self, 最小秒=0.5, 最大秒=1.5) -> None:
    """模拟真人操作的随机等待"""
    等待时间 = random.uniform(最小秒, 最大秒)
    await asyncio.sleep(等待时间)

async def 操作前延迟(self) -> None:
    """每次页面操作前的短延迟（0.3~0.8秒）"""
    await self.随机延迟(0.3, 0.8)

async def 操作后延迟(self) -> None:
    """每次页面操作后的延迟（0.8~2.0秒）"""
    await self.随机延迟(0.8, 2.0)

async def 页面加载延迟(self) -> None:
    """等待页面加载/跳转后的延迟（1.5~3.0秒）"""
    await self.随机延迟(1.5, 3.0)

3. 商品列表页每个原子方法加延迟（pages/商品列表页.py）
每个原子方法的模板改为：
async def 输入商品ID(self, 商品ID: str) -> None:
    await self.操作前延迟()
    for 选择器 in 商品列表页选择器.商品ID搜索框.所有选择器():
        try:
            输入框 = self.页面.locator(选择器).first
            await 输入框.click(timeout=10000)
            await self.随机延迟(0.2, 0.5)  # 点击后停顿一下再输入
            await 输入框.fill(商品ID)
            await self.操作后延迟()
            return
        except Exception:
            continue
    raise RuntimeError("商品ID输入框定位失败")
​
以下每个方法都要加：
导航到商品列表() → 导航完成后加 页面加载延迟()
输入商品ID() → 操作前延迟 + 输入后操作后延迟
点击查询() → 操作前延迟 + 点击后操作后延迟
等待搜索结果() → 结果出来后加 随机延迟(1.0, 2.0)
点击发布相似() → 操作前延迟 + 点击后操作后延迟
确认发布相似弹窗() → 操作前延迟 + 确认后 页面加载延迟()
