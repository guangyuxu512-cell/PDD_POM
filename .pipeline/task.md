### 功能 1：发布相似商品去掉关闭弹窗

文件：`tasks/发布相似商品任务.py`

在 `执行` 方法中，当前代码（大约在 wait_for_load_state 之后）：

```python
发布页对象 = 发布商品页(新页面)
await 发布页对象.关闭所有弹窗()
```

改为：**删掉 `await 发布页对象.关闭所有弹窗()` 这一行**。发布相似品页面正常不会出现弹窗，调用关闭弹窗反而会误删图片。

### 功能 2：限时限量自动创建——折扣兜底

文件：`backend/services/任务参数服务.py`

修改 `批次完成后创建后续任务` 方法：

当前逻辑：从成功记录的 params 中读折扣，找不到就 return 0。

改为：

1. 优先从成功记录的 params 中读折扣
2. 如果所有成功记录都没有折扣值，**仍然创建限时限量记录**，把折扣值设为 `None`
3. 打印日志：`[任务参数服务] 批次 {批次ID} 成功记录无折扣值，限时限量记录折扣为空，需手动补充`

同时修改 `限时限量任务.py`（如果需要）：

- 不改。当折扣为空时，`_读取折扣` 会抛 `ValueError("折扣不能为空")` → 任务失败并记录错误，用户可以在前端补上折扣值后重置重跑。
